## Описание

Приложение для управления товарами и заказами магазина игрушек с разграничением прав доступа.

## Стек

- Python 3.13+
- customtkinter - графический интерфейс
- SQLite - система управления базами данных
- Pillow - работа с изображениями
- openpyxl - импорт данных из Excel

## Установка и запуск

### 1. Требования

- Python 3.10 или выше
- pip (менеджер пакетов Python)

### 2. Установка зависимостей

```bash
    pip install -r requirements.txt 
```
###    3. Запуск приложения
    bash
    python main.py
    При первом запуске база данных database.db создаётся автоматически.

###    4. Импорт данных
    После запуска приложения используйте кнопки в интерфейсе администратора:

#### Импорт товаров - загружает товары из data/товар.xlsx

#### Импорт пользователей - загружает пользователей из data/пользователи.xlsx

#### Импорт заказов - загружает заказы из data/заказ.xlsx

    Порядок импорта:

    Сначала товары (чтобы загрузились артикулы и поставщики)

    Затем заказы (будут подтягиваться товары по артикулам)

    Пользователи (можно в любой момент)

###   Учётные данные для входа
####    Администратор
    Логин	Пароль
    admin	admin123
####    Менеджер
    Логин	Пароль
    manager	manager123
    Авторизованный клиент
    Логин	Пароль
    client	client123
####    Гость
    Нажмите кнопку "Продолжить как гость"

###    Функциональность
####    Общие возможности (все роли + гость)
    Авторизация по логину и паролю

    Вход как гость (только просмотр товаров)

    Просмотр списка товаров в виде карточек

####    Цветовая маркировка товаров
    Условие	Цвет фона
    Нет в наличии (количество = 0)	Голубой (#ADD8E6)
    Скидка > 17%	#FFDEAD
####   Отображение цены со скидкой
    Старая цена - зачёркнутая красная

    Новая цена - жирная чёрная

###    Для Менеджера и Администратора
    Поиск товаров в реальном времени (по названию, категории, описанию, производителю)

    Сортировка по цене (возрастание/убывание) и количеству на складе

    Фильтрация по поставщику (выпадающий список)

    Просмотр заказов (таблица с артикулами, статусами, датами)

###    Только для Администратора
#####    Управление товарами
    Добавление новых товаров

    Редактирование существующих товаров

    Загрузка изображений (ограничение 300x200 пикселей)

    Удаление товаров (с проверкой: нельзя удалить товар, который есть в заказах)

#####    Управление заказами
    Добавление новых заказов с выбором товаров и количества

    Редактирование заказов (все поля, включая состав)

    Удаление заказов

    Просмотр состава заказа

#####    Импорт данных из Excel
    Импорт товаров (артикулы, наименования, цены, скидки, фото)

    Импорт пользователей (роли, ФИО, логины, пароли)

    Импорт заказов (номера, товары, даты, статусы, адреса)

#    База данных
##    Таблицы
    Таблица	Описание
    users	Пользователи (id, login, password, full_name, role)
    suppliers	Поставщики (id, name)
    products	Товары (id, article, name, category, description, manufacturer, supplier_id, price, unit, quantity, discount, image_path)
    orders	Заказы (id, article, status, pickup_address, order_date, issue_date)
    order_products	Связь заказов и товаров (order_id, product_id, quantity)
##   Связи
    products.supplier_id → suppliers.id

    order_products.order_id → orders.id

    order_products.product_id → products.id

##    Сборка исполняемого файла
```    bash
    pip install pyinstaller
    pyinstaller --noconfirm --windowed --onefile --name ToysShop --add-data "windows;windows" --add-data "data;data" --add-data "images_source;images_source" main.py 
```
    Исполняемый файл будет в папке dist/

    База данных database.db создаётся автоматически при первом запуске
